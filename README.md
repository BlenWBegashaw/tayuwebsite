# TAJU Heavy Equipment Rental Website

Link: https://taju.vercel.app

This is the official website for TAJU Business PLC, a heavy equipment rental service based in Addis Ababa, Ethiopia.

## 🚜 About
The website showcases available machinery for mining, quarry, and construction projects. Customers can view equipment and send booking or inquiry requests.

## Features
- Equipment listing page
- Contact form (Formspree integration)
- Mobile responsive design
- Clean modern UI using Tailwind CSS

## 📩 Booking Process
1. Customer selects equipment
2. Clicks "Get Quote"
3. Fills out contact form
4. TAJU team responds with availability and pricing

## 🛠 Tech Stack
- HTML
- Tailwind CSS
- JavaScript
- Formspree (for contact form)
